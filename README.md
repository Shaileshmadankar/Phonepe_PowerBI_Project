# PhonePe Power BI Dashboard

## Project Overview

**Project Title**: PhonePe Power BI Dashboard  
**Level**: Beginner  
**Tool**: Power BI  

This project demonstrates end-to-end data analysis using Power BI. It includes data cleaning, data modeling, and a fully interactive dashboard analyzing PhonePe’s digital payments across India.  

This project is ideal for beginners in data analytics and Power BI learners.

---

## Objectives

1. **Import & Clean Data**  
   - Download Excel files containing transactions, payments, transaction IDs, loans, recharge, etc.
   - Remove null/duplicate records, rename columns, convert data types, and clean unwanted values.

2. **Data Modeling**  
   - Create relationships between tables.  
   - Build a proper star schema or snowflake schema if required.

3. **Build Interactive Dashboard**  
   - Identify growth patterns, state-wise performance, and category-wise transactions.

4. **Generate Insights**  
   - Use SQL or DAX to answer specific business questions and derive actionable insights.

5. **Design a Professional Report**  
   - Include slicers, charts, bar graphs, pie charts, and other visuals for meaningful analysis.

---

## Project Structure

### 1. Data Cleaning (Power Query)
- Remove null or duplicate records  
- Rename columns for clarity  
- Convert data types appropriately  
- Clean unwanted or incorrect values  

### 2. Data Modeling
- Connect tables using relationships  
- Create star or snowflake schema if required  

### 3. Dashboard Pages

#### **Home Page**
- Cards for: Total Transactions, Successful Transactions, Total Amount, Failed Transactions  
- Date range filter  
- Bar graph for Service vs Amount  

#### **Insurance Page**
- Date range filter  
- Total insurance amount by category (Bike, Car, Term Life, Health)  
- Payment status and failed payment status (pie charts)  
- Bar graph for vehicle insurance  
- Month-wise insurance amounts  

#### **Loan Page**
- Date range filter  
- Total loan amount by category (Gold Loan, Auto Loan, Mutual Fund, Credit Card Loan)  
- Month-wise loan distribution  
- Failed loan amounts  

#### **Money Transfer Page**
- Date range filter  
- Sum of money transfer by card  
- Transaction count by card  
- Payment status (pie chart)  
- Type of money transfer (Bar Graph: UPI ID, QR Code, Account, Mobile Number)  
- Month-wise transfer amount (Area Chart)  

#### **Recharge & Bills Page**
- Date range filter  
- Total recharge & bills amount by category (Electricity, DTH, Mobile, Water, Cable TV)  
- Payment status and failed payment status (Pie Chart)  
- Month-wise amount (Area Chart)

## Insights & Findings

- **Home Page**: Complete summary of UPI transactions, total amount, total transactions, overall payments including insurance and loans.  
- **Insurance Page**: Insights on insurance transactions by category, month, and payment status.  
- **Loan Page**: Loans distributed by category and month, with failed loan tracking.  
- **Money Transfer Page**: Patterns in person-to-person and person-to-merchant transfers, monthly trends, and payment modes.  
- **Recharge & Bills Page**: Insights into utility payments and recharges, identifying peak months and category-wise distribution.  

**Key Takeaway:**  
PhonePe’s digital payments dashboard empowers businesses to make data-driven decisions, maximize profits, and minimize losses by analyzing user behavior, transaction patterns, and regional performance.
